# LightPage

一个不好看的个人主页。无聊随便做的。

## 特色

1. 简单的页面
2. 显示 Typecho 文章（RSS）
3. 亮色/暗黑模式
4. ~~二次元~~

## 演示

https://lazy.ink

## 使用方法

1. 克隆本仓库
2. 修改相关信息
3. 上传到 ~~你的网站服务器~~ [Edgeone Pages](https://edgeone.ai/pages/new?repository-url=https%3A%2F%2Fgithub.com%2FLazyCreeper%2FLightPage&repository-name=LightPage&project-name=LightPage)
